# Security

Coursework code and data for ELEC0138 Security and Privacy.

## Main folders

- `Assessment/week4-vm-lab/`: cw1 and cw2 demo code, Snort rules, and sample outputs
- `Week 4 - Attacks Mitigation and Data Security/`: input PCAP files used for replay

## Quick start

For the runnable demo, see:

- `Assessment/week4-vm-lab/README.md`

The lab is intended to run inside an Ubuntu VM on a private or host-only network.
